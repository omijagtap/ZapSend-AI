# **App Name**: ZapSend AI Email Dispatcher

## Core Features:

- CSV Data Import: Allows users to import contact and data from a CSV file.
- Email Template Upload: Users can upload a .txt file containing the email template with placeholders.
- Template Preview and Editor: Display how the email look with Upload Text editor adn with placeholders replaced with data from the CSV, with options to bold text and add attachments.
- Personalized Email Generation: Generates personalized emails for each recipient by replacing placeholders in the template with data from the CSV.
- Email Sending via SMTP: Send emails via a configured SMTP server, providing settings for the server, port, and credentials.
- Reporting and Logs: Generates a report with details of sent, failed, and skipped emails, which would include an explanation of reason and download status for each email send.
- Subject Line Optimization Tool: Leverages generative AI tool to suggest and optimize email subject lines for higher open rates.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to represent sophistication and professionalism, in line with ZapSend AI's brand.
- Background color: Very dark gray (#121212) for a dark theme, almost black but not quite.
- Accent color: Soft lavender (#D0BCFF) as a contrasting accent to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk' sans-serif for headings, giving a tech-forward but readable appearance. Body Font: Inter, sans-serif. 
- Logo: "ZapSend AI" in bold with brand color #F23E36.
- Fixed header at the top with the logo, ensuring constant brand visibility.
- Subtle animations for loading states and email status updates to enhance user experience.
- Use clean, minimalist icons to represent different functions like upload, send, and report generation.
